## Portfolio Overview

This portfolio is organized as a connected system of repositories:

**Signal → Thinking → Tools → Build**

- **Signal** → curated market intelligence and external inputs  
- **Thinking** → mental models and decision frameworks  
- **Tools** → AI models, tools, and product capabilities  
- **Build** → experiments, prototypes, and applied execution  

This structure reflects how modern product leaders learn, decide, and operate.
