## Portfolio Operating System

A structured operating system for governing product and technology portfolios, integrating strategy decomposition, capital allocation, portfolio scoring, risk governance, and executive decision support.

Repository:

https://github.com/ChuckFerrando/enterprise-product-operating-model

Key components include:

- Capital allocation model for disciplined investment decisions
- Quantitative portfolio scoring framework
- Defense-grade risk evaluation model
- Portfolio heatmap visualization framework
- Governance cadence for portfolio execution
- Executive investment memo and decision brief templates
- Decision log for traceable governance
- AI-assisted portfolio scenario modeling
